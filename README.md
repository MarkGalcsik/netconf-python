# netconf-python

A script felhasználó által megadott NetConf szerverhez csatlakozikés és lehetővé teszi különböző konfigurációs lekérdeezések és mentések végrehajtását.

A megoldás a Cisco DevNet Sandboxban (Cisco IOS-XE alapú) lett tesztelve.

Funkciók: 
-Csatlakozás NetConf szerverhez felhasználó által megadott adatokkal 
-A szerver által támogatott funckiók/képességek (capabilities) lekérdezése 
-Teljes running konfiguráció lekérdezése 
-Szűrt konfiguráció lekérdezése filterek használatával 
-A running konfiguráció mentése XML fájlba
